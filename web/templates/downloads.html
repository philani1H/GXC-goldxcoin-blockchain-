Installation Instructions

Windows Installation
  1. Download the miner ZIP file for your chosen algorithm
  2. Extract the ZIP file to a folder (e.g., C:\GXC\Miners\)
  3. Double-click install.bat to install
  4. The installer will:
    • Copy files to %LOCALAPPDATA%\GXC\Miners\
    • Create a desktop shortcut
    • Add a start menu entry
  5. Edit config.json to set your wallet address: 
    {
  "pool_url": "{% if stratum_urls %}{{ stratum_urls.general_pool }}{% else %}stratum+tcp://pool.gxc.network:3333{% endif %}",
  "wallet_address": "YOUR_GXC_ADDRESS_HERE",
  "worker_name": "worker1",
  "threads": 4
}
  6. Launch the miner from your desktop or start menu
  7. Start mining!
Linux Installation
  1. Download the miner tar.gz file
  2. Extract the archive: 
    tar -xzf sha256_miner_gui-linux-x64.tar.gz
  3. Make the AppRun executable: 
    chmod +x sha256_miner_gui.AppDir/AppRun
  4. Create config file: 
    nano ~/.config/GXC/config.json
    Add: 
    {
  "pool_url": "{% if stratum_urls %}{{ stratum_urls.general_pool }}{% else %}stratum+tcp://pool.gxc.network:3333{% endif %}",
  "wallet_address": "YOUR_GXC_ADDRESS_HERE",
  "worker_name": "worker1",
  "threads": 4
}
  5. Run the miner: 
    ./sha256_miner_gui.AppDir/AppRun
  6. Start mining!

Getting Your Wallet Address

You need a GXC wallet address to receive mining rewards:

  1. Visit the Wallet API
  2. Register an account
  3. Create a new wallet
  4. Copy your GXC address (starts with "GXC")
  5. Save your private key and mnemonic phrase securely!
Important: Always backup your private key and mnemonic phrase! Without them, you cannot recover your wallet.
