Quick API Reference
Register User
POST http://localhost:5000/api/v1/register
Body: {"username", "email", "password"}
Login
POST http://localhost:5000/api/v1/login
Body: {"username", "password"}
Create Wallet
POST http://localhost:5000/api/v1/wallets/create
Headers: Authorization: Bearer TOKEN
Body: {"wallet_name", "password"}
Get Wallets
GET http://localhost:5000/api/v1/wallets
Headers: Authorization: Bearer TOKEN
Create Account
Quick Start: Register a new account to start using GXC wallets.
Option 1: Use API (For Developers)
POST http://localhost:5000/api/v1/register
Body: {
"username": "your_username",
"email": "your_email@example.com",
"password": "your_password"
}
Option 2: Use Wallet Dashboard

Visit the Wallet Dashboard to register and create your first wallet through the web interface.

Open Wallet Dashboard
Create Wallet
Authentication Required: You must be logged in to create a wallet.
Step 1: Login
POST http://localhost:5000/api/v1/login
Body: {
"username": "your_username",
"password": "your_password"
}
Response: { "token": "JWT_TOKEN", ... }
Step 2: Create Wallet
POST http://localhost:5000/api/v1/wallets/create
Headers: Authorization: Bearer <JWT_TOKEN>
Body: {
"wallet_name": "My Wallet",
"password": "wallet_password"
}
Or Use Wallet Dashboard

Visit the Wallet Dashboard to create your wallet through the web interface.
