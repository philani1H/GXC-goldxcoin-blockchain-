API Documentation

RESTful API endpoints for accessing GXC blockchain data

Base URL
http://localhost:3000
Available Endpoints
GET /api/stats

Get current network statistics including total blocks, transactions, addresses, and mining metrics.

curl http://localhost:3000/api/stats
Response: 
{
  "total_blocks": 1234,
  "total_transactions": 5678,
  "total_addresses": 890,
  "total_supply": 31000000,
  "hash_rate": 125.5,
  "difficulty": 1500.25,
  "avg_block_time": 2.5
}
GET /api/blocks?limit=10

Get recent blocks from the blockchain. Limit parameter (1-100) controls how many blocks to return.

curl "http://localhost:3000/api/blocks?limit=10"
Response: 
[
  {
    "number": 1234,
    "hash": "0xabc123...",
    "timestamp": "2024-01-15T10:30:00",
    "miner": "GXC1a2b3c...",
    "tx_count": 5,
    "reward": 2.5,
    "consensus_type": "pow"
  },
  ...
]
GET /api/transactions?limit=20

Get recent transactions from the blockchain. Limit parameter (1-100) controls how many transactions to return.

curl "http://localhost:3000/api/transactions?limit=20"
Response: 
[
  {
    "hash": "0xdef456...",
    "block": 1234,
    "from": "GXC1a2b3c...",
    "to": "GXC4d5e6f...",
    "value": 100.5,
    "fee": 0.01,
    "status": "success",
    "timestamp": "2024-01-15T10:30:00",
    "type": "transfer"
  },
  ...
]
GET /api/charts?days=14

Get historical chart data for analytics. Days parameter controls the time range (default: 14 days).

curl "http://localhost:3000/api/charts?days=14"
Response:

Integration Examples

            

            

                
JavaScript/TypeScript

                

                    
                    
// Fetch network stats
async function getNetworkStats() {
  const response = await fetch('http://localhost:3000/api/stats');
  const stats = await response.json();
  console.log('Total blocks:', stats.total_blocks);
  return stats;
}

// Fetch recent blocks
async function getRecentBlocks(limit = 10) {
  const response = await fetch(`http://localhost:3000/api/blocks?limit=${limit}`);
  const blocks = await response.json();
  return blocks;
}

                


                
Python

                

                    
                    
import requests

# Get network stats
def get_network_stats():
    response = requests.get('http://localhost:3000/api/stats')
    return response.json()

# Get recent blocks
def get_recent_blocks(limit=10):
    response = requests.get(f'http://localhost:3000/api/blocks?limit={limit}')
    return response.json()

# Example usage
stats = get_network_stats()
print(f"Total blocks: {stats['total_blocks']}")
