Mining Guide

Complete instructions, downloads, and setup guide for GXC blockchain mining

Quick Start (5 Minutes)

1
Download Mining Software

Choose your platform and download the appropriate miner:

Linux x64 Windows x64 macOS x64 All Downloads
2
Create Configuration File

Create a mining configuration file:

# Create config directory mkdir -p ~/.gxc # Create mining config cat > ~/.gxc/mining.conf << EOF algorithm=gxhash threads=0 address=GXC1YOUR_MINER_ADDRESS_HERE pool=stratum+tcp://pool.gxc.network:3333 username=your_worker_name password=x EOF
3
Start Mining

Run the miner with your configuration:

# Linux/macOS ./gxc-gxhash-miner -c ~/.gxc/mining.conf # Windows gxc-gxhash-miner.exe -c %APPDATA%\GXC\mining.conf
4
Monitor Your Mining

Check your mining stats:

  • Pool Dashboard: Coming Soon - Pool dashboard will be available at http://localhost:3001
  • Command Line: gxc-gxhash-miner --stats
  • Explorer: View Network Mining Stats

Configuration Examples

CPU Mining (GXHash) Configuration
# GXHash CPU Mining Configuration algorithm=gxhash threads=0 # 0 = auto-detect optimal thread count address=GXC1YOUR_MINER_ADDRESS_HERE # Pool Mining (Recommended) pool=stratum+tcp://pool.gxc.network:3333 username=your_worker_name password=x # Advanced Options thread_affinity=0,1,2,3,4,5,6,7 huge_pages=true intensity=75
GPU Mining (Ethash) Configuration
# Ethash GPU Mining Configuration algorithm=ethash devices=0,1,2,3 # GPU device IDs intensity=85 address=GXC1YOUR_ADDRESS # Pool configuration pool=stratum+tcp://gpu-pool.gxc.network:4444 username=your_worker_name.gpu password=x # Performance tuning (NVIDIA) core_clock=+100 # MHz offset memory_clock=+500 # MHz offset power_limit=250 # Watts fan_speed=80 # Percentage
ASIC Mining (SHA-256) Configuration
# ASIC Configuration for GXC SHA-256 mining algorithm=sha256 pool=stratum+tcp://asic-pool.gxc.network:3333 username=your_worker_name.asic password=x address=GXC1YOUR_ADDRESS # ASIC-specific settings frequency=650 # MHz voltage=0.9 # Volts fan_speed=80 # Percentage
Note: Most ASIC miners can be configured via their web interface. Access your miner at http://MINER_IP_ADDRESS and configure the pool settings.
