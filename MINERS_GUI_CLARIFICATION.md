# GXC Miners - GUI Status

## ✅ Miners WITH GUI

1. **gxc-mining-gui** (C++ Qt GUI) ⭐
   - Full-featured mining GUI
   - Qt-based graphical interface
   - Platform: Windows, Linux, macOS

2. **gxc-wallet** (C++ Qt GUI)
   - Wallet application with mining tab
   - Includes mining functionality
   - Platform: Windows, Linux, macOS

3. **gxc-node-gui** (C++ Qt GUI)
   - Node management GUI
   - Blockchain monitoring
   - Platform: Windows, Linux, macOS

4. **gxhash_miner.py** (Python Tkinter GUI) ⭐
   - Python miner with GUI
   - Tkinter-based interface
   - Platform: Windows, Linux, macOS

## ❌ Miners WITHOUT GUI (Command-Line Only)

1. **gxc-miner** - Universal miner (CLI)
2. **gxc-sha256-miner** - SHA-256 miner (CLI)
3. **gxc-ethash-miner** - Ethash miner (CLI)
4. **gxc-gxhash-miner** - GXHash miner (CLI)
5. **gxc-pool-proxy** - Pool proxy (CLI)

## Summary

- **4 miners have GUI** (gxc-mining-gui, gxc-wallet, gxc-node-gui, gxhash_miner.py)
- **5 miners are CLI only** (gxc-miner, gxc-sha256-miner, gxc-ethash-miner, gxc-gxhash-miner, gxc-pool-proxy)

All are installable, but only some have GUIs.
