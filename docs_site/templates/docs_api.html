API Documentation

RPC API

GXC blockchain provides a JSON-RPC API on port 8545.

Blockchain Methods

curl -X POST http://localhost:8545 \\ -H "Content-Type: application/json" \\ -d '{ "method": "gxc_getBlockchainInfo", "params": [], "id": 1 }'

Mining Methods

# Get block template curl -X POST http://localhost:8545 \\ -d '{"method": "gxc_getBlockTemplate", "params": [{"algorithm": "sha256"}], "id": 1}' # Submit block curl -X POST http://localhost:8545 \\ -d '{"method": "gxc_submitBlock", "params": [block_data], "id": 1}'

REST API

Explorer provides REST endpoints:

  • {{ explorer_url }}/api/blocks - Get blocks
  • {{ explorer_url }}/api/transactions - Get transactions
  • {{ explorer_url }}/api/stats - Get network stats
