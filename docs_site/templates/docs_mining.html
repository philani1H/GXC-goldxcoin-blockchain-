Mining Guide

Complete guide to mining on the GXC blockchain

Mining Algorithms

SHA-256 (ASIC)

Bitcoin-compatible ASIC mining

Stratum URL: stratum+tcp://asic-pool.gxc.network:3333

Dashboard: asic-pool.gxc.network

Ethash (GPU)

Ethereum-compatible GPU mining

Stratum URL: stratum+tcp://gpu-pool.gxc.network:4444

Dashboard: gpu-pool.gxc.network

GXHash (CPU/All)

Custom CPU-optimized algorithm, supports all algorithms

Stratum URL: stratum+tcp://pool.gxc.network:3333

Dashboard: pool.gxc.network

Quick Start

For ASIC Miners (SHA-256)

# Configure your ASIC miner # Pool URL: stratum+tcp://asic-pool.gxc.network:3333 # Username: your_worker_name # Password: x # Algorithm: SHA-256

For GPU Miners (Ethash)

# Download GPU miner # Visit http://localhost:3000/downloads or use: curl -O http://localhost:3000/api/downloads/linux/ethash tar -xzf ethash_miner_gui-linux-x64.tar.gz # Configure cat > mining.conf << EOF algorithm=ethash pool=stratum+tcp://gpu-pool.gxc.network:4444 username=your_worker password=x address=GXC1YOUR_ADDRESS EOF # Start mining ./gxc-miner -c mining.conf

For CPU/General Mining (GXHash)

# Download miner # Visit http://localhost:3000/downloads or use: curl -O http://localhost:3000/api/downloads/linux/gxhash tar -xzf gxhash_miner_gui-linux-x64.tar.gz # Configure cat > mining.conf << EOF algorithm=gxhash pool=stratum+tcp://pool.gxc.network:3333 username=your_worker password=x address=GXC1YOUR_ADDRESS EOF # Start mining ./gxc-miner -c mining.conf

Pool Configuration Examples

BFGMiner / CGMiner (ASIC)

bfgminer -o stratum+tcp://asic-pool.gxc.network:3333 -u your_worker -p x

Ethminer / T-Rex (GPU)

ethminer -P stratum1+tcp://your_worker:x@gpu-pool.gxc.network:4444

GXC Miner (CPU/General)

./gxc-miner --pool stratum+tcp://pool.gxc.network:3333 --user your_worker --pass x
