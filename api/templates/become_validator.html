Using a Third-Party Wallet?

If you're using an external wallet (not created on this platform), you can still register as a validator:

  • Make sure your wallet has the required stake amount
  • The system will automatically try external validator registration if needed
  • For API users: Use POST /api/v1/external-validator/register
  • Or import your private key via POST /api/v1/wallet/import-key
