QUICK FIX TO BUILD THE BLOCKCHAIN:

1. Download SQLite3 manually:
   - Go to: https://www.sqlite.org/download.html
   - Scroll to "Precompiled Binaries for Windows"
   - Download: "sqlite-dll-win64-x64-*.zip" OR "sqlite-tools-win-x64-*.zip"
   - OR download "sqlite-amalgamation-*.zip" (source code version)

2. Extract and place files:
   - If you downloaded the DLL version:
     * Extract sqlite3.h from the zip
     * Place it in: third_party\sqlite3\sqlite3.h
     * You'll need sqlite3.lib (compile it or get precompiled)
   
   - If you downloaded amalgamation (RECOMMENDED):
     * Extract sqlite3.h and sqlite3.c
     * Place both in: third_party\sqlite3\
     * Files should be: third_party\sqlite3\sqlite3.h and third_party\sqlite3\sqlite3.c

3. Build:
   cmake -S . -B build
   cmake --build build --config Release

ALTERNATIVE: Use vcpkg or chocolatey:
   choco install sqlite
   Then set: cmake -DSQLite3_ROOT_DIR=C:\ProgramData\chocolatey\lib\sqlite ..
